Image

Images make the website more interactive, as they display more context to the user.

Square Image

Square images as the name suggest are squared with a default dimension of 150*150.

Rounded Image

Rounded images as the name suggest are round with a default dimension of 150*150.

Responsive Image

Responsive Images are images which fit to the parent component. They expand and shrink upto to the width of the container.

Avatar

Avatars are images that users can set as their profile picture.

Profile Avatar

Profile Avatars can be created using the class 'avatar', inside the img tag.

Avatar Sizes

Size of the avatars can be changed by adding avatar-sm | avatar-md | avatar-lg class to the existing avatar class.

Square Avatar

Shape of the avatars can be changed by adding square class to the existing avatar class.
